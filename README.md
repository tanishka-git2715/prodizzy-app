# Prodizzy Platform

This is the unified mobile and web application for Prodizzy, built with Next.js.

## Getting Started

To run this project locally, you need to have **Node.js** installed on your computer.

### 1. Install Dependencies

Open your terminal (Command Prompt, PowerShell, or VS Code Terminal) in this folder and run:

```bash
npm install
```

### 2. Run the Development Server

Once the installation is complete, start the app with:

```bash
npm run dev
```

### 3. View the App

Open your web browser and go to:

[https://www.prodizzy.com/](https://www.prodizzy.com/)
